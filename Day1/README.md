# Day 1 - Express Server Basics

## Overview
Introduction to Express.js and basic server setup.

## What's Covered
- Installing Express using npm
- Creating a basic Express server
- Starting a server on port 3000 using `app.listen()`

## Technical Concepts
- **Express.js**: Minimal and flexible Node.js web application framework
- **Server Initialization**: Using `express()` to create an application instance
- **Port Binding**: Listening on a specific port to handle incoming requests

## Run
```bash
npm install
node app.js
```
