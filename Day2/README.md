# Day 2 - HTTP Routing with GET Requests

## Overview
Understanding HTTP routing and handling GET requests in Express.

## What's Covered
- Creating multiple routes (`/`, `/about`, `/contact`)
- Using `app.get()` to handle GET requests
- Sending responses with `res.send()`

## Technical Concepts
- **Routing**: Defining how an application responds to client requests at specific endpoints
- **HTTP GET Method**: Used for retrieving data from the server
- **Request-Response Cycle**: Understanding req and res objects

## Run
```bash
npm install
node app.js
```
Visit `http://localhost:3000`, `http://localhost:3000/about`, or `http://localhost:3000/contact`
