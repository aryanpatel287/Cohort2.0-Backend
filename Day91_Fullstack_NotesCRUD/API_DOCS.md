# API Documentation — Notes CRUD

**Base URL (Development):** `http://localhost:3000`  
**Base URL (Production):** `https://cohort2-0-backend-z7gf.onrender.com`

All request and response bodies use **JSON**. Set the `Content-Type: application/json` header on all requests with a body.

---

## Endpoints Overview

| Method   | Endpoint           | Description          |
|----------|--------------------|----------------------|
| `POST`   | `/api/notes`       | Create a new note    |
| `GET`    | `/api/notes`       | Get all notes        |
| `GET`    | `/api/notes/:id`   | Get a note by ID     |
| `PATCH`  | `/api/notes/:id`   | Update a note        |
| `DELETE` | `/api/notes/:id`   | Delete a note        |

---

## POST `/api/notes`

Create a new note.

### Request

**Headers**
```
Content-Type: application/json
```

**Body**
```json
{
  "title": "Meeting Notes",
  "description": "Discuss project timeline and deliverables."
}
```

| Field         | Type   | Required | Description          |
|---------------|--------|----------|----------------------|
| `title`       | String | Yes      | Title of the note    |
| `description` | String | Yes      | Body of the note     |

### Response

**Status:** `201 Created`

```json
{
  "message": "Notes created successfully",
  "note": {
    "_id": "64f1a2b3c4d5e6f7a8b9c0d1",
    "title": "Meeting Notes",
    "description": "Discuss project timeline and deliverables.",
    "__v": 0
  }
}
```

---

## GET `/api/notes`

Retrieve all notes.

### Request

No body or query parameters required.

### Response

**Status:** `200 OK`

```json
{
  "message": "Notes fetched successfully",
  "notes": [
    {
      "_id": "64f1a2b3c4d5e6f7a8b9c0d1",
      "title": "Meeting Notes",
      "description": "Discuss project timeline and deliverables.",
      "__v": 0
    },
    {
      "_id": "64f1a2b3c4d5e6f7a8b9c0d2",
      "title": "Shopping List",
      "description": "Milk, eggs, bread.",
      "__v": 0
    }
  ]
}
```

Returns an empty array `[]` under `notes` if no notes exist.

---

## GET `/api/notes/:id`

Retrieve a single note by its MongoDB `_id`.

### Request

**URL Parameter**

| Parameter | Type   | Description              |
|-----------|--------|--------------------------|
| `id`      | String | MongoDB ObjectId of note |

**Example:**
```
GET /api/notes/64f1a2b3c4d5e6f7a8b9c0d1
```

### Response

**Status:** `200 OK`

```json
{
  "message": "Notes fetched successfully",
  "notes": {
    "_id": "64f1a2b3c4d5e6f7a8b9c0d1",
    "title": "Meeting Notes",
    "description": "Discuss project timeline and deliverables.",
    "__v": 0
  }
}
```

---

## PATCH `/api/notes/:id`

Partially update an existing note. Only the fields provided in the body are updated; omitted fields retain their current values.

### Request

**URL Parameter**

| Parameter | Type   | Description              |
|-----------|--------|--------------------------|
| `id`      | String | MongoDB ObjectId of note |

**Headers**
```
Content-Type: application/json
```

**Body** *(all fields optional)*
```json
{
  "title": "Updated Title",
  "description": "Updated description text."
}
```

| Field         | Type   | Required | Description                           |
|---------------|--------|----------|---------------------------------------|
| `title`       | String | No       | New title. Keeps existing if omitted  |
| `description` | String | No       | New body. Keeps existing if omitted   |

**Example — update title only:**
```json
{
  "title": "Revised Meeting Notes"
}
```

### Response

**Status:** `200 OK`

```json
{
  "message": "Note updated successfully",
  "updatedNote": {
    "_id": "64f1a2b3c4d5e6f7a8b9c0d1",
    "title": "Revised Meeting Notes",
    "description": "Discuss project timeline and deliverables.",
    "__v": 0
  }
}
```

## DELETE `/api/notes/:id`

Delete a note by its MongoDB `_id`.

### Request

**URL Parameter**

| Parameter | Type   | Description              |
|-----------|--------|--------------------------|
| `id`      | String | MongoDB ObjectId of note |

**Example:**
```
DELETE /api/notes/64f1a2b3c4d5e6f7a8b9c0d1
```

### Response

**Status:** `200 OK`

```json
{
  "message": "Note deleted successfully"

```

## Data Model

```
Note {
  _id          : ObjectId   (auto-generated by MongoDB)
  title        : String
  description  : String
  __v          : Number     (Mongoose version key)
}
```

---

## Error Handling

> The current implementation does not include explicit error responses. The following are the common failure scenarios:

| Scenario                        | Likely Outcome                             |
|---------------------------------|--------------------------------------------|
| Invalid / malformed `id`        | Mongoose `CastError` — server crashes or hangs (no try/catch) |
| Note not found for `GET /:id`   | Returns `null` under `notes` key           |
| Note not found for `PATCH /:id` | Mongoose `findById` returns `null` — server crashes |
| Missing required fields on POST | Note created with `undefined` fields       |
| MongoDB connection failure      | Server fails to start                      |

## Example Usage with cURL

**Create a note**
```bash
curl -X POST https://cohort2-0-backend-z7gf.onrender.com/api/notes \
  -H "Content-Type: application/json" \
  -d '{"title":"Buy Groceries","description":"Milk, eggs, bread"}'
```

**Get all notes**
```bash
curl https://cohort2-0-backend-z7gf.onrender.com/api/notes
```

**Get a note by ID**
```bash
curl https://cohort2-0-backend-z7gf.onrender.com/api/notes/64f1a2b3c4d5e6f7a8b9c0d1
```

**Update a note**
```bash
curl -X PATCH https://cohort2-0-backend-z7gf.onrender.com/api/notes/64f1a2b3c4d5e6f7a8b9c0d1 \
  -H "Content-Type: application/json" \
  -d '{"title":"Updated Title"}'
```

**Delete a note**
```bash
curl -X DELETE https://cohort2-0-backend-z7gf.onrender.com/api/notes/64f1a2b3c4d5e6f7a8b9c0d1
```

## Example Usage with Axios (Frontend)

```js
import axios from 'axios'

const api = axios.create({ baseURL: 'http://localhost:3000' })

// Create
await api.post('/api/notes', { title: 'New Note', description: 'Some text' })

// Read all
const { data } = await api.get('/api/notes')
console.log(data.notes)

// Read one
const { data } = await api.get('/api/notes/64f1a2b3c4d5e6f7a8b9c0d1')

// Update
await api.patch('/api/notes/64f1a2b3c4d5e6f7a8b9c0d1', { title: 'Updated' })

// Delete
await api.delete('/api/notes/64f1a2b3c4d5e6f7a8b9c0d1')
```
